# Background Videos

The weather background video files are **not included** in this repository to keep it lightweight.

The app works fine without them — it will simply skip the video background.

To enable dynamic video backgrounds, add your own `.mp4` files here with these exact names:

| Filename            | Weather condition          |
|---------------------|----------------------------|
| `clear.mp4`         | Clear sky                  |
| `clouds.mp4`        | Cloudy                     |
| `rain.mp4`          | Rain / drizzle             |
| `snow.mp4`          | Snow                       |
| `thunderstorm.mp4`  | Thunderstorm               |
| `mist.mp4`          | Mist / fog / haze / dust   |
